ch_pipeline.processing.daily.csds_in_range(start, end, step=1)[source]

Get the CSDs within a time range.

The start and end parameters must either be strings of the form “CSDd+” (i.e. CSD followed by an int), which specifies an exact CSD start, or a form that ephemeris.ensure_unix understands.

Parameters:

  • start (str or parseable to datetime) – Start of interval.

  • end (str or parseable to datetime) – End of interval. If None use now. Note that for CSD intervals the end is inclusive (unlike a range).

Returns:

csds

Return type:

list of ints
