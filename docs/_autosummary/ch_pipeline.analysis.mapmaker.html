ch_pipeline.analysis.mapmaker¶

Map making tasks

Tools for making maps from CHIME data.

RingMapMaker()

A simple and quick map-maker that forms a series of beams on the meridian.
class ch_pipeline.analysis.mapmaker.RingMapMaker[source]¶

Bases: draco.core.task.SingleTask

A simple and quick map-maker that forms a series of beams on the meridian.

This is designed to run on data after it has been collapsed down to non-redundant baselines only.

npix¶

Number of map pixels in the declination dimension. Default is 512.

Type

int

span¶

Span of map in the declination dimension. Value of 1.0 generates a map that spans from horizon-to-horizon. Default is 1.0.

Type

float

weight¶
How to weight the non-redundant baselines:

‘natural’ - each baseline weighted by its redundancy (default) ‘uniform’ - each baseline given equal weight ‘inverse_variance’ - each baseline weighted by the weight attribute

Type

string (‘natural’, ‘uniform’, or ‘inverse_variance’)

exclude_intracyl¶

Exclude intracylinder baselines from the calculation. Default is False.

Type

bool

include_auto¶

Include autocorrelations in the calculation. Default is False.

Type

bool

single_beam¶

Only calculate the map for the central beam. Default is False.

Type

bool

Initialize pipeline task.

May be overridden with no arguments. Will be called after any config.Property attributes are set and after ‘input’ and ‘requires’ keys are set up.
